Portifólio

Você pode acompanhar um pouco dos meus projetos aqui pelo meu repositório do Github, lá você vai encontrar alguns projetos usados para fins de estudo bem básicos até um projeto mais completo em fase de finalização um aplicativo chamado Movies Review.

O app consiste em um aplicativo de avaliações e indicações de filmes.
Estou usando como base de estudos o desenvolvimento deste app usando Flutter no front e Backend usando Firebase da Google.
Usei uma API gratuita disponibilizada pela comunidade do TMDB , onde tem um repositório de filmes com imagens e todas as informações do mesmo.
Neste aplicativo você vai poder escolher o filme, avaliar dando estrelas que vão de 0 a 5 estrelas, com isso é feito uma média de avaliações e é dada a nota do filme, suas avaliações ficam visíveis para todos os usuários para assim, se quiserem, debater sobre.

Ele ainda não está pronto, tenho que finalizar algumas coisas e colocar na PlayStore, mas você pode ver esse projeto aqui; Movies Review
